Sistema de Gestão de Alunos
Sistema web desenvolvido em PHP para gerenciamento de alunos, permitindo cadastrar, editar, listar e excluir registros.

Funcionalidades
Listar Alunos: Visualização de todos os alunos em tabela.
Cadastrar Aluno: Formulário para inserir novo aluno e matrícula.
Alterar Aluno: Edição de dados existentes.
Excluir Aluno: Remoção de registros do banco de dados.

Tecnologias Utilizadas
Linguagem: PHP (Orientado a Objetos).
Banco de Dados: MySQL.
Front-end: HTML.

Estrutura dos Arquivos
conexaoAlunos.php: Classe de conexão e manipulação do banco de dados.
verAluno.php: Página principal com listagem e processamento de ações.
alterarAluno.php: Formulário de edição de alunos.
addAluno.php: Formulário de cadastro de novos alunos.
